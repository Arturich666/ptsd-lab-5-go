# Book Management REST API

REST API для управління каталогом книг, написаний на мові **Go**. Проект реалізовано з дотриманням принципів чистішої архітектури, де логіка обробки HTTP-запитів повністю відокремлена від шару збереження даних за допомогою інтерфейсів.

---

## API Ендпоінти

| Метод | Шлях | Опис |
| :--- | :--- | :--- | 
| `GET` | `/books` | Отримати список усіх наявних книг | 
| `GET` | `/books/{id}` | Отримати детальну інформацію про книгу за її UUID | 
| `POST` | `/books` | Створити нову книгу | 
| `PUT` | `/books/{id}` | Повне оновлення даних книги за ID | 
| `PATCH` | `/books/{id}` | Часткове оновлення окремих полів книги | 
| `DELETE` | `/books/{id}` | Видалити книгу з бази даних | 

---
# Запуск

Введіть в терміналі:
```
go mod tidy
go run main.go
```

Після цього перейдіть на `http://localhost:8080`
---
# Запуск unit-тестів
Введіть в терміналі:
```
go test ./... -v
```

# Валідація даних
- Title і Author не можуть бути порожніми
- Year має бути більшим за 0 і меншим за поточний рік
- Price має бути додатнім числом
